<task type="auto">
  <name>Task 4: Create mobile menu JavaScript</name>
  <files>wp-content/plugins/artricenter-structure/assets/js/mobile-menu.js</files>
  <action>
Create mobile-menu.js with:

1. **Menu toggle functionality**:
   ```javascript
   (function() {
       'use strict';

       const menuToggle = document.querySelector('.artricenter-menu-toggle');
       const menuClose = document.querySelector('.artricenter-menu-close');
       const menuOverlay = document.querySelector('.artricenter-menu-overlay');
       const body = document.body;

       // Open menu
       if (menuToggle) {
           menuToggle.addEventListener('click', function() {
               menuOverlay.classList.add('artricenter-menu-overlay--open');
               menuToggle.setAttribute('aria-expanded', 'true');
               body.style.overflow = 'hidden'; // Prevent background scrolling
           });
       }

       // Close menu
       if (menuClose) {
           menuClose.addEventListener('click', function() {
               menuOverlay.classList.remove('artricenter-menu-overlay--open');
               menuToggle.setAttribute('aria-expanded', 'false');
               body.style.overflow = '';
           });
       }

       // Close on backdrop click
       if (menuOverlay) {
           menuOverlay.addEventListener('click', function(e) {
               if (e.target === menuOverlay) {
                   menuOverlay.classList.remove('artricenter-menu-overlay--open');
                   menuToggle.setAttribute('aria-expanded', 'false');
                   body.style.overflow = '';
               }
           });
       }

       // Close on Escape key
       document.addEventListener('keydown', function(e) {
           if (e.key === 'Escape' && menuOverlay.classList.contains('artricenter-menu-overlay--open')) {
               menuOverlay.classList.remove('artricenter-menu-overlay--open');
               menuToggle.setAttribute('aria-expanded', 'false');
               body.style.overflow = '';
           }
       });
   })();
   ```

2. **Submenu accordions** (if menu has children):
   - Add click listeners to submenu toggle buttons

   - Update aria-expanded attributes

3. **Accessibility**:

   - Escape key closes menu
   - Body scroll lock when menu open

4. **No jQuery dependency**:
   - Use vanilla JavaScript
   - No external dependencies

Use IIFE to avoid global namespace pollution. Add comments for accessibility features.

   ```php
    * Retrieve the Artricenter header HTML.
    *
    * @return string Header HTML markup

   function artricenter_get_header(): string {
       $header = new \Artricenter\Structure\Header();
       return $header->render();
   }

   /**
    * Display the Artricenter header.
    *
    * Usage in theme: <?php artricenter_the_header(); ?>
    */
   function artricenter_the_header(): void {
       echo artricenter_get_header();
   }
   ```

2. **Enqueue JavaScript** (hook: wp_enqueue_scripts):
   ```php
   function artricenter_enqueue_scripts(): void {
       wp_enqueue_script(

           array(),                    // No dependencies
   ```
